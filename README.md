
### Features

- Check Website info
- Check Phone info
- IP Tracker
- Check Valid E-mail
- Check if site is Up/Down
- Check internet speed
- Check Personal info
- Find IP behind Cloudflare
- Find Subdomains
- Port Scan (Multi-threaded)
- Check CMS
- Check DNS leaking


### Usage:
```
cd infog
bash infog.sh
```

### Install requirements (Curl, Netcat):

```
apt-get install -y curl nc
```
